# 🚀 Space Shooter AI & Analytics

A complete **Game Development + Data Science Project** built using Python.

This project combines a **Pygame-based space shooter game** with a **Streamlit analytics dashboard** to analyze gameplay data.

---

## 🎮 Game Features

- Smooth player movement (Arrow keys)
- Continuous shooting system (SPACE key)
- Enemy spawning with increasing difficulty
- Health bar ❤️ and lives system
- Score & Level tracking
- Sound effects 🔊 (laser shooting)
- Explosion effect 💥
- Pause (P key) & Restart (R key)
- Game Over screen

---

## 🎯 Controls

| Action | Key |
|--------|-----|
| Move | Arrow Keys ⬅️➡️⬆️⬇️ |
| Shoot | SPACE 🔫 |
| Pause | P |
| Restart | R |

---

## 📊 Data Science Integration

- Gameplay data stored in `game_data.csv`
- Data includes:
  - Score
  - Level
  - Lives
  - Timestamp

---

## 📈 Dashboard (Streamlit)

- Score trend visualization 📊
- Level vs Score analysis
- Average score calculation
